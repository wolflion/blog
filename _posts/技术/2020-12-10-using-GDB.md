+ 2、调用gdb的三种方法

  + （3）gdb正在运行的进程
    + gdb pid
    + gdb attach pid
    + gdb -p pid

### 二、gdb的相关命令

+ 比较常用：
  + `b`：设置断点 break  文件名:行号
  + `d`：删除断点
  + `p`：打印   p 变量名
  + `s`：进入函数内部
  + `bt`：进入堆栈，**但只显示10层**。最内层n，最外层-n
+ 其它（我不太熟悉的）
  + `f`：结束函数 finish
  + `u`：结束循环  until
  + `info local`：所有的局部变量。**只有debug版本才行**。
  + `thread`：线程
    + `thread apply all bt`：**所有线程**（查看死锁时使用）
  + `frame`：
  + `x`：内存地址
  + `info register`：寄存器
  + `watch`：显示内存
