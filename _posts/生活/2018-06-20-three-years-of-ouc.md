## 二、硕士基本流程

### 2.1、考试&入学

#### 2.1.1、前期

+ 作为一个江苏人知道OUC，其实是很偶然，一来呢感谢研招网的分类查找，二来呢当时身边正好有人是这个学校的。
+ 既然定好学校了，信院又有我要的专业，于是就买来真题，当时也没跟人提起，就自己闲下来的时候自己复习。
+ 其实都没太怎么看，但学校的真题可是好好做了的。*英语和政治*差不多佛系吧，考前一周进行了调休，当时在南工大（丁家桥）复习的，中午有时就去蹭主任食堂的饭。**当时还行，没有赶人，也算是个安静时光**

#### 2.1.2 考试

+ 考试在我家附近，早上骑个电瓶车就去了（我还保留了当时考点的照片，当然是考后才拍的）。
+ D1考完两门并没有啥感觉，因为关机，考完开机了发现晚上还有约饭的，就骑车去了河西万达，吃完饭后还跟（小北河、榴莲）他们掼蛋到22点才回的。
+ D2早上起来后继续考，现在已经不记得啥状态了。只知道考完了回家拎个箱子就出差去了。
+ 日子一天天过，成绩出来了，本来我觉得我考330+不算太高，但一看排名，觉得稳了。
+ 进入复试名单时，当时也不知道联系谁，也没加群，后面遇到**宝罗**了，一打电话就是河南话，也不清楚他说的啥。毕竟复试也不敢太马虎（英语+笔试+面试）呢，根据大纲和要求准备呗。**甚至也没有提前联系导师**。
+ 复试的时候，当时住的青旅，那时机票好便宜（单程才200块左右），也没太紧张，比较内向，也没跟人太多寒暄，去青旅跟人扯淡了。
+ D3天晚上的飞机，当时还去青岛博物馆逛了逛，在逛的时候说是出录取名单了，开手机共享和电脑，查看了一下，虽然是意料之中的排名，但毕竟官宣。**当时也没心情逛博物馆了，感觉蛮爽**。
+ 在酒店门口坐大巴到机场，整了一碗70的面，晚上还有人接机，去瑞金路吃了顿干锅牛蛙（好久没吃干锅牛蛙啦）。
+ 这都6月份左右了，人家说导师都确定了，我就咱导写了个邮件（当时复试我的组长就是他，还问我的导师找了谁，我说没找呢，他说有没有意向的，我说的就是他，还顺带夸了一顿）

#### 2.1.3 入学

+ 一直工作到开学前一天才歇，同时办了助学贷款。
+ 入学的时候到没啥感觉，差点没赶上去青岛的高铁，到了青岛后东西太多（70L背包、箱子、还有个拎包）叫了个黑车去的学校。
+ 当时没找到9号楼，还是个小姐姐带我去的，（给我推销被子啥的，我还是买的官方的，搞得我都有点不好意思）。
+ 安顿OK，30号咱导给我写邮件了，进入垃圾邮箱了，我也没看到，比较傻，31号才去找导师，幸好还没满，一届6个，最后一个是我。*当时董主席也在找他*。

### 2.2、上课&论文

#### 2.2.1、上课

+ 修学分嘛，都是正常的教学流程，我竟然**自然辩证法**挂科了，考了64还是多少的，当时老师给了打印材料，自己不想背，画了思维导师啥的，成绩出来傻眼了，70分才算及格。
+ 魏振刚老师就是PPT老师了，不过选他课的人贼多，我好像还选了两门，都偏软件工程的课。
+ 最坑的是杨永全老师的《云计算》，我当时可很想学的，第一堂课上的激昂慷慨，第二次我带了个装着ubuntu的系统准备大展拳脚的时候，最后所有的课程都不上了。**最后一门考试竟然就是考第一次的内容**。
+ 英语是我比较怂的课，英语有挂科（实验室师兄有挂科），有平时分+考试分+口语分（representation+dialog）但我最后70分过了，**估计老师给我放水了**。
+ 研一上我差不多把课都修光了，本来专硕课程和学分就不算多。
+ 剩下的就是**实验室时光**了

#### 2.2.2、论文

+ 研二论文开题，差不多就是个流程了，根据你的平时项目以及导师安排，安排个论文开题和中期审核（有时开题都省了）。不过，这个和最终的论文其实关系也没有那么大。
+ 研三找完工作，差不多最晚11月底得开始写论文了，写论文跟写作一样的，你要每天给自己任务，一天得有个1000字左右，当然架不住，导师看了一遍后，这一节都不要了，白瞎了。
+ 我觉得还是要早点写好，导师看了好有个数。**我的都是要交外审了，导师才给突击了一下中英文摘要**。
+ **查重**
  + 学校查重呢，如果不严的话，可以不在乎，但在那个氛围下，钱都不重要了，哪怕一个人也要查一下知网，毕竟花钱买心安啊。
  + 我这个金牛座嘛，把钱不当钱当命，然后就没查，最后重复率6还是8吧。
+ **外审**
  + 外审这个事吧，是不是院长以下的都操作不了啊，就等着，**我交完外审，去西北浪了半个月吧**。
  + 最后两个B的结果，从全班返回结果来看，其实18年情况要比17年要好很多。
+ **答辩**
  + 本来以为答辩是走过场，最后全院都开始紧张起来，包括我们导师，我们导师强烈觉得我的论文有可能会被现场专家给问死，让我哑口无言。
  + 我按导师的意思又跑了几个用例，最后还行，答辩专家没有为难我。
  + **不过咱导一早7点就问我在哪一组，专家有谁，我答辩过程中也问，过程怎么样**，赶紧给他回一个，感觉这是导师对自己最好的一天了，至少是一天之一。（聚餐的时候也好，还给夹菜，哈哈）
  + 作为班长，有给自己排哪一组的权利，最后发现专硕都挺OK的，最后我们答辩10人组还跟专家愉快的合个影，有个别女专家有点扭捏，但架不住大势所趋，还是从了。

+ 9月来实验室，第一学期就是上课和熟悉环境。（*也培训了ExtJS*，但自己没好好学，20170719觉得**这是自己不想逃离舒适区**。）

### 3.3、导师

+ 项目上的事就不说了，说说我日常的糗事吧。

#### 3.3.1、开会绑架导师请吃饭

+ 应该是16年5月左右，做完常规的年终汇报，已经过了11点饭点了，好像还有一两个没汇报，
  + 导师：“食堂没饭吃了吧，要不下午再讲”。
  + 我：“不不不，讲完咱们去南区吃”。
  + 导师：“南区没位置吧”
  + 我：“不不不，有位置”。
  + .....吃完.....
  + 我：”我付钱“
  + 导师：”我来，今天不贵，吃得挺好“

#### 3.3.2、被导师问Jerry在不在

+ 这个很多人都知道，不提了

#### 3.3.3、被开摄像头

+ 当时校外有个大数据培训，要5天的样子，跟导师请假，导师不给我假，我就自己直接去了，第三天被逮到了。先是电话跟我说事的，我心慌说了两件给挂了。**Je老rry就说了，你可牛逼了，你是第一个敢挂导师电话的**。
  + 我：”老师，我跟我同学去看脚去了“。
  + 导师：”那你把医院的就诊证明，拍给我看看“
  + 一看瞒不住，导师：”那你开个摄像头吧，让我知道你在哪“  
  + **回去啊，被他们一顿嘲讽，还全员开会批评，罚了一个月俸禄**。

#### 3.3.4、故意严肃

+ 我们外审的时候，导师也方，跟我说，论文外审”我比较担心你的，还有谁的“。最后出结果的时候，我跟他汇报时。
  + 我：”祝徐老师和徐老师朋友的学生都过了“（伸手准备去握手）
  + 导师：（顿时收起还在笑的脸）”这有啥恭喜的，这不都是应该的嘛“。

## 四、业余生活

+ 当时是15年3月左右报的，已经公示录取了就报了HJ商学院，5月开学，虽然是青岛班吧，但之前都在南京上过1次课，有1次还给错过了。

+ 真正转入青岛了，每月虽然线下进行听课，也认识一些朋友，参与了一些元旦、端午的活动。

+ 2016年5月31日被A6录取，每周10公里的跑步（*除了有次跑步摔跤后，有两周没跑，其余每周都跑*），于是有了后面跑半马的记录。
